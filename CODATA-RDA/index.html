NOTE: how to build list of ‘##’ headers from lesson-01.md

In the example I used grep to find the ‘##’ lines, then sed to clean up some text, and finally loop through each line and display the results with a ‘ + ’ prefix.

    grep -E '^##' lesson-01.md | \
    sed -e "s/\*//g" -e "s/##//g" -e "s/^\s|\s$//" | \
    while read LINE; do 
        echo "    + $LINE"; 
    done
